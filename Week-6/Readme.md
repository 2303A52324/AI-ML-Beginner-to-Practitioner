# Student Grade Prediction Flask App

This project predicts a student's final grade using Machine Learning.

## Technologies
- Python
- Flask
- Scikit-learn
- Pandas
- HTML
- CSS

## Steps

1. Install requirements

pip install -r requirements.txt

2. Train Model

python train_model.py

3. Run Flask

python app.py

4. Open

http://127.0.0.1:5000
